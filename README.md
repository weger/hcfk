# hcfk
highlight code for keynote

## Installing
````javascript
$ [sudo] npm install hcfk -g
````

## Basic Usage
````
// for mac
$ CMD + C    // copy your code

$ hcfk

$ CMD + V

````

- [brew]()
- [highlight](http://www.andre-simon.de/doku/highlight/en/highlight.php)
